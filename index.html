Memrise Review Calculator

This is a tool designed to show you how many flash card reviews you can expect per day using Memrise's Spaced Repetition System. The SRS algorithm was taken from the Memrise website message boards.

The current graph is calculated using 20 new flash cards per day over the course of 90 days.

To see the code, or to download so you can use the command line tool, please check out [the github repo](https://github.com/xferguson/memrise-src).
